# Cosplay ITA Blog 🌟

Blog italiano dedicato al mondo del cosplay. Articoli approfonditi, guide tecniche, foto AI esclusive e news dalla community italiana.

## Caratteristiche

- Design colorato e moderno con tema scuro
- 3 articoli lunghi con 3 foto AI ciascuno
- Ottimizzato per la SEO (sitemap, robots.txt, meta tag)
- Aggiornamento automatico ogni 3 giorni tramite GitHub Actions
- Responsive per mobile e desktop

## Struttura

```
cosplay-ita-blog/
├── index.html          # Homepage
├── articolo1.html      # Anime & Fiere
├── articolo2.html      # Crafting & Tutorial
├── articolo3.html      # Makeup & Styling
├── sitemap.xml         # SEO Sitemap
├── robots.txt          # SEO Robots
├── assets/
│   ├── css/style.css   # Stile principale
│   ├── js/main.js      # JavaScript
│   └── images/         # Foto AI
└── .github/workflows/  # GitHub Actions
```
